== Description ==

The Buzzzy button is the best way for your visitors to share your blog posts with the Google Buzz community. 

At Buzzzy.com we track all the shares going through Google Buzz so this button will display the count of who's actually sharing your posts.


= Features =

* Live count of Buzz's on Google Buzz
* Simple to install and customise
* Position before or after your blog posts.
* Position anywhere manually within your post with the buzzzy shortcode
* View a list of who's buzzing your blog posts
* Very fast download time.
* Allow readers to simply Buzz your blog post without leaving your site.


== Installation ==

Follow the steps below to install the plugin.

1. Upload the Buzzzy directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/Buzzzy to configure the button
